## Usage

### Script

```js
import { Array } from 'library-math';
```

###

### reducer method

```js
Array.reduce([1, 2, 3]);
```

```js
Array.reduce([1, 2, 3], 5);
```

###

### concat method

```js
Array.concat([1, 2, 3], [4, 5, 6]);
```

###

### join method

```js
Array.join([1, 2, 3]);
```

### filter method

```js
Array.filter([1, 2, 3], 1);
```
